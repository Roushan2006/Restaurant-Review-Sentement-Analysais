# 🍽️ Restaurant Review Sentiment Analysis

## 📌 Overview
Restaurant Review Sentiment Analysis is a project that uses Natural Language Processing (NLP) to evaluate customer reviews and classify them as Positive, Negative, or Neutral. It helps restaurant owners understand customer feedback and improve service quality.

---

## 🚀 Features
- 🔍 Analyze customer reviews automatically  
- 😊 Classify sentiment (Positive, Negative, Neutral)  
- 📊 Simple and user-friendly system  
- ⚡ Fast processing of large datasets  
- 🧠 Helps in better decision-making  

---

## 🛠️ Technologies Used
- Python  
- NLTK 
- Pandas  
- Matplotlib

## ⚙️ Installation

1. Clone the repository:

git clone https://github.com/Roushan2006/restaurant-sentiment-analysis.git


2. Navigate to the project folder:

cd restaurant-sentiment-analysis


3. Install required libraries:

pip install -r requirements.txt


---

## ▶️ Usage

Run the program:

python main.py


Example:

Review: The food was delicious!
Sentiment: Positive
